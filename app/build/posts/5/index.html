A 4-Part Series on Building Apps with CanJS
  • canjs
  • javascript

by alex 3 days ago

I started using CanJS while I was working on a shipping application at a logistics company that serves most of the world. CanJS was the bedrock for their real-time dashboard application, and for good reason: it offers a variety of observable data structures as well as computed properties, so you can keep your UI in sync with a dynamic data collection. And it turns out that they aren't the only ones using it, the Apple Store is, too! All in all, CanJS is a very capable and progressive framework. However, I felt the need to write some tutorials because I found the documentation a bit lacking and scattered among a bunch of different locations on the internet.
